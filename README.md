# POTHUB
Website Platform edukasi budidaya tanaman

POTHUB adalah platform edukasi berbasis web yang menyediakan informasi mengenai berbagai jenis tanaman, khususnya untuk pemula yang ingin belajar budidaya. Website ini menampilkan katalog tanaman dalam bentuk kartu yang berisi gambar, nama, dan deskripsi singkat, serta fitur “Baca Lebih Lanjut” untuk melihat detail informasi. Selain itu, terdapat akses admin yang memungkinkan pengelolaan data tanaman secara CRUD (Create, Read, Update, Delete). Dengan tampilan yang sederhana dan ramah pengguna, POTHUB memudahkan pengguna dalam mengenal dan mempelajari berbagai jenis tanaman.
